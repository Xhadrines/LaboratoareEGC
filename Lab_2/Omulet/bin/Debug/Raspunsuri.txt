1. Ce este un viewport?
Este o fereastră în cadrul ferestrei principale unde se desenează scenele 3D.

2. Ce reprezintă conceptul de frames per seconds din punctul de vedere al bibliotecii OpenGL?
FPS (Frames Per Second) reprezintă numărul de cadre (imagini) afișate pe secundă și indică viteza de randare a scenei 3D în OpenGL. Cu cât FPS-ul este mai mare, cu atât randarea este mai fluentă și mai rapidă.

3. Când este rulată metoda OnUpdateFrame()?
Metoda OnUpdateFrame() este rulată în fiecare cadru în OpenGL, înainte de randarea scenei, pentru a actualiza starea aplicației, cum ar fi mișcarea obiectelor sau gestionarea inputului utilizatorului.

4. Ce este modul imediat de randare?
Modulul imediat de randare este o abordare mai veche în OpenGL în care se utilizează funcții precum glBegin() și glEnd() pentru a desena obiecte 3D direct pe ecran. Această abordare a fost înlocuită în versiunile moderne ale OpenGL deoarece este mai puțin eficientă și mai puțin flexibilă.

5. Care este ultima versiune de OpenGL care acceptă modul imediat?
Ultima versiune de OpenGL care acceptă modul imediat este OpenGL 3.0.

6. Când este rulată metoda OnRenderFrame()?
Metoda OnRenderFrame() este rulată în fiecare cadru, după ce metoda OnUpdateFrame() a fost apelată.

7. De ce este nevoie ca metoda OnResize() să fie executată cel puțin o dată?
Metoda OnResize() trebuie să fie executată cel puțin o dată pentru a gestiona redimensionarea ferestrei OpenGL și pentru a ajusta corect viewport-ul și proiecția astfel încât scena 3D să se potrivească cu dimensiunea ferestrei.

8. Ce reprezintă parametrii metodei CreatePerspectiveFieldOfView() și care este domeniul de valori pentru aceștia?
Metoda CreatePerspectiveFieldOfView() este folosită pentru a crea o matrice de proiecție perspectivică în OpenGL.
